# Desafios de Lógica de Programação

Este repositório contém desafios resolvidos em várias linguagens **(JS, Python, Java)**, organizados por **nível de dificuldade** e **plataforma** (LeetCode, Beecrowd, etc).

---

## Níveis de Dificuldade

- **Iniciante**: Problemas básicos com lógica simples e estruturas simples.
- **Intermediário**: Envolve estruturas de dados e algoritmos mais elaborados.
- **Avançado**: Algoritmos mais complexos, como grafos, programação dinâmica, entre outros.

---

## Estrutura do Projeto


Cada subpasta possui versões do mesmo problema resolvidas em diferentes linguagens.

---

## Exemplos de Desafios

### Iniciante
- Duas somas
- Adicione dois números
- Inverter uma string

### Intermediário
- Busca binária
- Fila com prioridade
- Algoritmo de ordenação (bubble, merge, quick)

### Avançado
- Dijkstra (menor caminho em grafo)
- Problema da mochila (knapsack)
- Sudoku solver

---

## Como Usar

Acesse a pasta de dificuldade e linguagem desejada, leia o enunciado no início do código e execute com seu interpretador/compilador.
